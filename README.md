# Swift Hackaton PJ-03

## An example of a simple banking application that supports the operations of replenishing a bank account, withdrawing cash from a bank account, replenishing a mobile phone account, and maintaining information by phone numbers.

### The application works with the Realm database.

### The application implements all the functions stated above, takes into account various situations and implements all checks for the erroneous use of user actions.

`Large files (more than 100 MB) of the Realm database with the ".a" extension were placed in the repository of the remote github repository using the GIT LFS framework.`
